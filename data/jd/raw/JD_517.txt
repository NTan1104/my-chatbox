Xây dựng hệ thống Chatbot Agent & RAG

Hỗ trợ xây dựng pipeline RAG (Retrieval-Augmented Generation) từ cơ bản: chunking tài liệu, tạo embedding, lưu trữ vào vector database, truy vấn và sinh câu trả lời.
Tham gia phát triển chatbot agent có khả năng hội thoại nhiều lượt, sử dụng tools và duy trì ngữ cảnh.
Tích hợp chatbot vào các nghiệp vụ thực tế của doanh nghiệp: kế toán, hóa đơn, chăm sóc khách hàng.
Viết code thử nghiệm (experiment) để đánh giá chất lượng kết quả trả về của hệ thống RAG.
Tham gia phát triển hệ thống Multi-Agent

Học và áp dụng kiến trúc multi-agent: agent điều phối (orchestrator), agent thực thi (executor), agent công cụ (tool agent).
Hỗ trợ viết và kiểm thử các luồng phối hợp giữa nhiều agent bằng LangChain, LangGraph hoặc framework tương đương.
Tích hợp các external API và công cụ (tools) vào hệ thống agent.
Hỗ trợ tích hợp & vận hành mô hình

Phối hợp với team backend và product để tích hợp tính năng AI vào sản phẩm nội bộ.
