Mô tả công việc:
I. MÔ TẢ CÔNG VIỆC- Lập trình phần server game cho game realtime (Phần server tương tự Clash Royale).- Thiết kế cơ sở dữ liệu cho sản phẩm.- Triển khai và vận hành Server.- Phát triển dịch vụ nền tảng và công cụ vận hành để xuất bản trò chơi.- Phát triển RESTful API dựa trên nền tảng và liên kết.- Phát triển các phân tích dữ liệu khác nhau và xử lý phân phối dữ liệu.- Phát triển các dịch vụ như quản lý thông tin, quản lý tài khoản và giám sát cần thiết cho các báo cáo thống kê.-Phát triển máy chủ phụ trợ cho các giao diện tích hợp nền tảng (cổng API, hàng đợi tin nhắn phân tán, v.v.)- Phát triển hệ thống xử lý thời gian thực cho trò chơi dữ liệu lớn.
